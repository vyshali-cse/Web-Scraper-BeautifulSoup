# Web Scraper using BeautifulSoup

## Description
A Python web scraper that extracts quotes from a website using BeautifulSoup.

## Features
- Sends HTTP requests
- Parses HTML content
- Extracts quotes from a webpage
- Displays scraped data

## Technologies Used
- Python
- Requests
- BeautifulSoup

## Files
- scraper.py
- requirements.txt

## How to Run

1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python scraper.py
